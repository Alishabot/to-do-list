# To-Do List CLI Application

A simple and interactive Command-Line Interface To-Do List application built with TypeScript and Node.js.

## Installation

```bash
git clone https://github.com/Alishabot/to-do-list.git
cd to-do-list
npm install
npm run build
```

## Usage

```bash
npm start
```

## Menu Options

1. Add task - Add a new task (max 100 characters)
2. View tasks - Display all tasks with numbers
3. Delete a task - Remove tasks by index or name
4. Exit - Close the application safely
