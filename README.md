# To-Do List CLI Application

A simple and interactive Command-Line Interface To-Do List application built with TypeScript and Node.js.

## Features

- Add Tasks - Easily add new tasks
- View Tasks - Display all tasks with clear numbering
- Delete Tasks - Remove tasks by index or name
- Exit Confirmation - Confirm before closing

## Requirements

- Node.js 14.0.0 or higher
- npm or yarn package manager

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Alishabot/to-do-list.git
cd to-do-list
```

2. Install dependencies:
```bash
npm install
```

3. Build the project:
```bash
npm run build
```

## Usage

Run the application:
```bash
npm start
```

## Menu Options

1. Add task - Add a new task (max 100 characters)
2. View tasks - Display all tasks with numbers
3. Delete a task - Remove tasks by:
   - Index (task number)
   - Name (search by text)
4. Exit - Close the application safely

## Technologies Used

- TypeScript 5.0+
- Node.js 14+
- readline-sync for user input
